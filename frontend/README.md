# Signal — Frontend (React + Vite)

Part of the Signal Product Analytics & Retention Intelligence project.
See the root `README.md` for the full architecture and run instructions.

## Quick start
```bash
npm install
cp .env.example .env   # points at the backend (default http://localhost:3001)
npm run dev            # http://localhost:5173
npm run build          # production build -> dist/
```

Three views: Case Study (narrative + Recharts), Dashboard (4 tabs), and
Ask Signal (natural-language-to-SQL chat, needs the backend running).
